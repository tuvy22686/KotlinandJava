## うれしいこと

- 日本語が使えて面白い
- publicがデフォ。わざわざ書かなくても良い

## かなしいこと

- Kotlinでは、変数宣言のとき型から書かない。よって、変数名を自動補完してくれないことがある
ex) Javaでは``Button bu(ここでtab)``→``Button button``になる  
しかし、Kotlinでは``val but(ウッ)``となる。(Buttonぐらいならしてくれるかも)
